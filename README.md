BoardZone is a modern web application built with **Next.js (App Router)** and **PandaCSS**, designed for easy online table reservations in a board-game bar. Users can pick a date, reserve a table, browse the menu, and check opening hours and contact details. The project focuses on clean UI, fast performance, and a simple reservation flow suitable for both desktop and mobile users.

---

## Tech Stack

- **Next.js 14+ (App Router)**
- **React**
- **PandaCSS** (design system & styling)
- **TypeScript**
- **NextAuth.js** (authentication)
- **Drizzle ORM + PostgreSQL** (optional, depending on project scope)
- **Vercel** (deployment)

---

## Installation

```bash
git clone https://github.com/<your-repo>/boardzone.git
cd boardzone
pnpm install
pnpm dev
```
